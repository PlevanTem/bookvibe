# BookVibe 快速开始指南

## 🚀 立即使用（最简单）

1. **直接打开 `index.html`**
   - 双击文件，在浏览器中打开
   - 输入书名，点击"出发"
   - 完成！

项目已预配置 API keys，无需任何设置即可使用。

## 📝 使用示例

### 测试书籍

尝试输入以下书名：

- **《挪威的森林》** - 村上春树
- **《了不起的盖茨比》** - F. Scott Fitzgerald
- **《百年孤独》** - 加西亚·马尔克斯
- **《追风筝的人》** - 卡勒德·胡赛尼
- **《解忧杂货店》** - 东野圭吾

### 功能说明

1. **输入书名** → 点击"出发"或按回车
2. **加载动画** → 显示"正在翻阅书籍..."和"正在寻找坐标..."
3. **明信片展示** → 点击卡片可翻转查看正反面
4. **换图片** → 点击"换一张图片"按钮
5. **保存图片** → 长按卡片 500ms 自动下载拼接长图

## ⚙️ 自定义配置

### 修改 API Keys

#### 方法 1：直接修改代码（最简单）

编辑 `app.js`，找到第 5-15 行：

```javascript
const CONFIG = {
    // 1. LLM API配置
    LLM_API_KEY: "你的-GLM-API-Key",
    LLM_MODEL: "GLM-4.7",
    
    // 2. 图片搜索API配置
    IMAGE_API_KEY: "你的-Unsplash-API-Key",
    // ...
};
```

#### 方法 2：使用配置文件

1. 复制 `config.example.js` 为 `config.js`
2. 编辑 `config.js`，填入你的 API keys
3. 在 `index.html` 中取消注释：

```html
<script src="config.js"></script>
```

### 获取 API Keys

#### GLM-4.7 API Key

1. 访问 [智谱AI开放平台](https://open.bigmodel.cn/)
2. 注册/登录账号
3. 创建应用，获取 API Key
4. 注意：需要充值才能使用

#### Unsplash API Key

1. 访问 [Unsplash Developers](https://unsplash.com/developers)
2. 注册/登录账号
3. 创建新应用
4. 复制 Access Key

## 🔧 故障排查

### 问题 1：API 调用失败

**症状**：显示"这本书太神秘，我们的旅行家迷路了。"

**解决方案**：
1. 检查 API keys 是否正确
2. 检查网络连接
3. 检查 API 余额（GLM 需要充值）
4. 查看浏览器控制台错误信息

### 问题 2：图片加载失败

**症状**：明信片正面没有图片

**解决方案**：
1. 检查 Unsplash API Key 是否正确
2. 检查是否超过免费额度（50 requests/hour）
3. 系统会自动使用备用图片

### 问题 3：CORS 错误

**症状**：浏览器控制台显示 CORS 错误

**解决方案**：
- 如果使用 GLM API，可能需要通过后端代理
- 参考 `api-example.js` 创建后端服务
- 或者使用支持 CORS 的 API 服务

## 🌐 部署到服务器

### 静态文件部署

1. **Vercel**（推荐）
   - 访问 https://vercel.com
   - 导入 GitHub 仓库
   - 自动部署

2. **Netlify**
   - 访问 https://netlify.com
   - 拖拽 `bookvibe` 文件夹
   - 完成部署

3. **GitHub Pages**
   - 推送到 GitHub
   - 在仓库设置中启用 Pages
   - 选择 `bookvibe` 目录

### 注意事项

⚠️ **安全提示**：
- 如果部署到公开网站，建议使用后端代理 API
- 不要在前端代码中暴露真实的 API keys
- 使用环境变量或后端服务保护 API keys
